# MH MARBLES | Visionary Architectural Gallery

A premium digital experience for Telangana's premier destination for exquisite marble, granite, and stones. Founded on a legacy of architectural excellence since 1980.

## ✨ Features

- **Cinematic Experience**: High-end GSAP and ScrollTrigger animations for a world-class architectural feel.
- **Signature Curation**: A meticulously curated masonry gallery showcasing the world's most exquisite stones.
- **Fluid Navigation**: Integrated Lenis smooth scrolling for a premium, momentum-based touch experience.
- **Bespoke Inquiries**: A personalized concierge interface for high-end architectural consultations.

## 🛠️ Technology Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom luxury Design System
- **Animations**: GSAP (GreenSock), ScrollTrigger, Framer Motion, Lenis
- **Tooling**: Vite for lighting-fast development
- **Icons**: Lucide React

## 🚀 Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📂 Project Structure

- `src/components`: Reusable luxury UI components (Header, Hero, ProductCollection, etc.)
- `src/pages`: Multi-page architecture (Curation, Legacy, Journal, Concierge)
- `src/lib`: Design system utilities, animations, and shared logic
- `src/index.css`: Global luxury palette (Obsidian & Radiant Gold)
